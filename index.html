合格版：複数記録 / CBT / 記録状況

ムード・トラッカー

1日に複数の記録を残せます。複数タグ、CBTメモ、追加項目、傾向別バー表示、CSV/JSONバックアップに対応しています。

記録状況

時間帯とタグを付けると、条件別の傾向を見返しやすくなります。

CBTメモ

状況・自動思考・適応思考を短く残せます。疲れている日は自由メモだけでも構いません。

表示する軸

見たい項目だけ表示できます。色だけでなく線のパターンでも区別します。

時系列グラフ

記録は日時順で表示します。外出記録は点線系列です。

曜日別傾向

曜日ごとの平均を簡易バーで表示します。

時間帯別傾向

朝 / 昼 / 夜での平均を簡易バーで表示します。

タグ分析

よく使うタグごとの傾向を簡易バーで表示します。

簡易相関メモ

追加項目が高い記録と低い記録で、関心の平均がどう動くかを見ます。

感情・状態の軸を追加する

例：不安、焦り、達成感、眠気、孤独感。

現在の追加項目

不要になった項目は削除できます。

タグテンプレート

入力タブに出すタグを追加・削除できます。

バックアップ

JSON で保存・復元できます。
