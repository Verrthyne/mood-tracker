# ムード・トラッカー 合格版

## 内容
このZIPは、会話内で合格版として扱うことになったムード・トラッカーです。

## 含まれるファイル
- index.html
- styles.css
- app.js
- README.md
- .nojekyll

## 主な仕様
- 1日に複数記録を保存可能
- 複数タグ対応
- 「記録状況」表記
- クイック入力 ON：スライダー群中心。記録状況とCBTメモは非表示
- クイック入力 OFF：記録状況とCBTメモを表示
- CBTメモ対応
- 曜日別 / 時間帯別 / タグ分析の簡易バー表示
- CSV / JSON エクスポート
- JSON インポート
- GitHub Pages 用の .nojekyll 同梱

## 使い方
GitHub Pages のリポジトリに以下を上書きアップロードしてください。

- index.html
- styles.css
- app.js
- .nojekyll

README.md は説明用なので、必須ではありません。
